# Chapter-4
第四章算例中涉及的数据与产消者参数

# 概述
本项目是博士论文 《智能配电网产消者能量共享优化方法研究》-“第四章 适应源荷不确定性的无迭代优化方法” 中案例研究参数的说明文档。

> _**2-node case**_： 2 节点下产消者的能量共享案例研究参数与场景数据集。  
> _**33-node case**_：33 节点下产消者的能量共享案例研究参数与场景数据集。  
> _**69-node case**_：69 节点下产消者的能量共享案例研究参数与场景数据集。   

每个文件夹中中含有静态文件参数，以及场景数据两种类型的文件，其中每个子文件夹的名称表示场景数据的随机程度（0.2即为20%的不确定性）

# 1. 静态参数文件

此类文件定义了模型的基础物理约束与经济边界：

* **`prosumer_parameters.xlsx`**
    * **内容**：各产消者（Prosumer）的独立决策参数。
    * **涵盖详情**：包含成本函数的系数、设备容量边界以及各类型分布式资源（光伏、储能、柔性负荷等）的运行限制与初始状态。
* **`community_parameters.xlsx`**
    * **内容**：产消者社区（Community）的全局拓扑与公共参数。
    * **涵盖详情**：包含电网各线路的流入流出节点及其电阻电抗参数。

# 2. 场景数据文件

此类文件用于多智能体值函数的学习，提供了不确定性环境的训练与测试数据样本。为便于调用，数据被精细拆分为三大类：

* **`prosumer_res_scenarios_train.xlsx`**
* **`prosumer_res_scenarios_verify.xlsx`**
    * **内容**：可再生能源（RES，如屋顶光伏或风电）出力的历史场景数据。
    * **结构**：内含各产消者`prosumer_i`的子表，分别记录了不同产消者在各个时序下的出力场景。
* **`prosumer_load_scenarios_train.xlsx`**
* **`prosumer_load_scenarios_verify.xlsx`**
    * **内容**：常规电力负荷（Load/Demand）的历史场景数据。
    * **结构**：内含各产消者`prosumer_i`的子表，分别记录了不同产消者在各个时序下的需求场景
* **`prosumer_price_scenarios_train.xlsx`**
* **`prosumer_price_scenarios_verify.xlsx`**
    * **内容**：外部公共电网电价波动的历史场景数据。
    * **结构**：包含 `buy`（向上层电网购电的价格）和 `sell`（向上层电网售电的价格）两个子表，记录了相关电价场景。
